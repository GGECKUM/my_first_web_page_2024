Software Engineering Degree - Year 3

The third year of the Software Engineering degree program focuses on advanced topics in software development and engineering. Students will learn about software design, web development, and software testing, among other subjects.

Courses

  • Software Design and Architecture
    • Learn about software design principles and architectural patterns.
    • Topics include design patterns, software architecture styles, and best practices.
  • Web Development
    • Introduction to web technologies and frameworks.
    • Learn about front-end and back-end development, including HTML, CSS, JavaScript, and server-side programming.
  • Software Testing and Quality Assurance
    • Study of software testing methodologies and techniques.
    • Learn about test planning, test case design, and automated testing.
  • Computer Networks
    • Introduction to computer networking concepts and protocols.
    • Topics include network architecture, TCP/IP, and network security.
  • Linear Algebra
    • Study of linear algebra concepts and their applications in computer science.
    • Topics include matrices, vectors, and linear transformations.
