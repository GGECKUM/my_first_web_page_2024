Software Engineering Degree - Year 1

The first year of the Software Engineering degree program provides students with a solid foundation in computer science and programming. The courses are designed to introduce fundamental concepts and skills that are essential for advanced studies in software engineering.

Courses

  • Introduction to Programming
    • Learn the basics of programming using languages such as Python or Java.
    • Understand fundamental programming concepts like variables, loops, and functions.
  • Discrete Mathematics
    • Study mathematical structures that are fundamental to computer science.
    • Topics include logic, set theory, combinatorics, and graph theory.
  • Computer Systems
    • Introduction to computer architecture and operating systems.
    • Learn about hardware components, memory management, and system software.
  • Introduction to Software Engineering
    • Overview of software development processes and methodologies.
    • Learn about requirements analysis, design, implementation, and testing.
  • Calculus I
    • Introduction to differential and integral calculus.
    • Topics include limits, derivatives, integrals, and applications of calculus.
