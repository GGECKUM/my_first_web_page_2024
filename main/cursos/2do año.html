Software Engineering Degree - Year 2

The second year of the Software Engineering degree program builds on the foundational knowledge gained in the first year. Students will delve deeper into programming, data structures, and algorithms, while also exploring new areas such as databases and operating systems.

Courses

  • Data Structures and Algorithms
    • Advanced study of data structures such as trees, graphs, and hash tables.
    • Algorithm design and analysis, including sorting, searching, and graph algorithms.
  • Object-Oriented Programming
    • Introduction to object-oriented programming concepts and principles.
    • Learn about classes, objects, inheritance, polymorphism, and encapsulation.
  • Database Systems
    • Introduction to database design and management.
    • Learn about SQL, relational databases, and database normalization.
  • Operating Systems
    • Study of operating system concepts and architecture.
    • Topics include process management, memory management, and file systems.
  • Calculus II
    • Continuation of Calculus I, covering advanced topics in differential and integral calculus.
    • Topics include sequences, series, and multivariable calculus.
